# Login

After you have installed Thor Sniper, it will open automatically. To start using it, you need to get your API key through our official Telegram bot.

**Generate Your Access Code & Login:**

1. Open the official Telegram bot using this [link](https://t.me/thorsolana_bot?start=r-mmdoseh-lextlogin).&#x20;

{% embed url="https://t.me/thorsolana_bot?start=r-mmdoseh-lextlogin" %}

2. Copy the login code from the Telegram message, paste it into the Thor Web Extension, and click Verify.

{% hint style="info" %}
**Note: Y**ou can click **Get Code** directly inside the Desktop App. This code is required to securely connect Thor Sniper to your account.
{% endhint %}
