# Activation

### Activation Steps

{% stepper %}
{% step %}
Click on Get Code.

<div align="left"><figure><img src="../../.gitbook/assets/image (13) (2).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}

<div align="left"><figure><img src="../../.gitbook/assets/image (8).png" alt="" width="391"><figcaption></figcaption></figure></div>
{% endstep %}

Copy and paste the code into the extension.

<div align="left"><figure><img src="../../.gitbook/assets/image (15) (1).png" alt=""><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Click Verify.

<div align="left"><figure><img src="../../.gitbook/assets/image (16) (2).png" alt=""><figcaption></figcaption></figure></div>


{% endstep %}

{% endstep %}

{% step %}

{% endstep %}
